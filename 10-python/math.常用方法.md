# number 常用方法


```python
import math

```

## ceil方法

> 向上取整


```python

print(math.ceil(123.223))
```

    124


## floor

> 向下取整


```python
print(math.floor(4.9))
```

    4


## round

> 四舍五入


```python
print(round(4.834, 2))

```

    4.83


## pow

> 指数计算


```python
print(pow(2,10))
```

    1024


## abs

> 绝对值


```python
print(abs(-10))
```

    10


## max 、 min

> max: 最大值
> min: 最小值


```python
print(max(1,10))

print(min(1,10))
```

    10
    1
